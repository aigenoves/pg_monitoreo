## Requirements
* Install Docker

## Not Essential 
* Install GNU Make (Not essential)

## Use the following command to run
* Run `docker compose up` (If you installed makefile you can use `make up`)


## Ports
The containers and their exposed ports are:

-   **postgresql** - `:5432`
-   **prometheus** - `:9090`
-   **grafana** - `:3000`
-   **postgresql-exporter** - `:9187`
